# Personal Finance Tracker

A simple web application to track personal expenses, built with **Python** and **Flask**.

The application allows users to register expenses, view recent transactions, generate a monthly report by category, and export the full history as a CSV file.

This project was initially generated using an AI-assisted workflow and is being used as a study project to explore backend development concepts.

---

## Features

- Add new expenses
- View recent expense history
- Monthly expense report by category
- Pie chart visualization
- Export all expenses to CSV
- SQLite database storage

---

## Tech Stack

- Python
- Flask
- SQLite
- Jinja2 Templates
- Chart.js
